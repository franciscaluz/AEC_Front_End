## POUR PUBLIER ONLINE
    * 1- changer sur le package.json:
      * "private": false,
      * "homepage": "./",
    * 2- changer sur le gitignore:
        * # production
            #/build
    * 3- yarn build
    * 4- commit + push

 * chaque changement n'est pas live, tu dois commit/push pour visualiser les changements
